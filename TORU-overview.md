# TORU — 建設現場向け図面解析アプリ

## プロジェクト概要

建設現場の施工図面（PDF）をAI解析し、材料拾い出し・見積書作成を自動化するアプリ。
配管業からスタートし、他職種へ展開する。

## コアバリュー

- 若手が図面を自力で読めるようになる
- 材料拾い出し → 見積書まで一気通貫で自動化
- 会社ごとの単価で正確な見積もりを作れる
- 一人親方からチーム会社まで、規模に応じた使い方ができる

## 機能定義

### MVP v0.1（1-2ヶ月で最速売り）

**「経営者が一番悩んでいる見積書機能」で先に価値を届ける。**

- 単価マスタ管理（会社別、手動入力 + **CSV/Excelインポート**）
- 手動材料入力 → 見積書PDF出力
- 共通マスタ（公開単価データ）参照・自社マスタへコピー
- ユーザー認証・会社アカウント・課金（Stripe）
- 業種タグ（プロフィール登録）
- **Web版先行リリース** → モバイル版並行開発

### MVP v0.2（+1-2ヶ月）

- 図面PDF解析（Claude API）→ 材料自動拾い出し
- 拾い出し結果 × 単価マスタで見積もり自動生成
- アプリ内課金（RevenueCat）追加

### MVP v0.3（+2-3ヶ月）

- プロジェクト/現場グループ機能（**法人プラン限定の作成権限**）
- **信頼グループタグ**（trusted/mixed）で共有時の警告レベル調整
- グループ共有（確認画面方式 → `group-share-confirmation-spec.md`）
- **招待リンク/QRコード + グループID手入力**のハイブリッド招待
- 法人請求書払い対応
- DM機能（必要に応じて）

## 機能詳細

### 1. 図面PDF解析（v0.2）
- 入力: ベクターPDF（CAD書き出し）。最大40ページ・13.52MB級まで想定
- ハイブリッド処理: 端末でテキスト抽出 → サーバー（Supabase Edge Functions）でClaude API
- 配管 v1 抽出項目:
  - 材料拾い出し（管種・径・数量）
  - 施工時の注意事項
  - 壁のふかし
  - 勾配情報
  - 他業者との緩衝箇所

### 2. 単価マスタ管理（v0.1）
- 各会社が独立した単価マスタを保有（厳密に分離）
- 入力項目: 材料名 / 単位 / 単価
- 共通マスタからコピーして使い始められる
- 詳細: `unit-price-master-spec.md`

### 3. 見積書作成（v0.1）
- 材料 × 単価マスタ → 自動計算 → PDF出力
- 経営者・現場監督向け

### 4. プロジェクト/現場グループ（v0.3）
- 複数会社・個人が横断参加できるグループ
- グループ作成: 現場監督が基本（個人/法人どちらでも可）
- 共有OK: 図面PDF / 構造データ（金額抜き）/ 施工メモ
- 共有NG（デフォルト警告）: 単価マスタ / 見積書 / 金額付き拾い出し
- **PDF添付時に3段階の確認画面**でオーナーが共有可否を判断

## マルチテナント設計

```
個人アカウント（一人親方）
  └─ 自分の単価マスタ / 見積書 / 図面（個人所有）

会社アカウント（複数メンバー、単価マスタ共有・厳密分離）
  └─ 会社の単価マスタ / 見積書 / メンバーの図面は個人単位で所有

プロジェクト/現場グループ（複数会社・個人が横断参加）
  ├─ 作成: 法人プラン限定
  ├─ 参加: 個人プランも可（招待経由）
  ├─ 信頼フラグ: trusted（自社グループ・身内）/ mixed（同業他社混在）
  ├─ 共有OK: 図面PDF / 構造データ（金額抜き）/ 施工メモ
  └─ 共有時要確認: 単価/見積書（3段階確認画面）
```

**図面の所有権:** アップロードした個人ユーザーが所有。グループ共有は本人判断で自由。
