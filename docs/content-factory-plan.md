## 1. ハヤカワ五味式ワークフローの分解

> 「ネタ指定（自動化可能）→ 台本作成 → 私っぽい口語に直す → TTSで音声化 →
> 冒頭は実写のリップシンクで動画生成、後半はClaudeCodeが動画生成 → 統合して完成」

| # | 工程 | 担当 | Claude Codeで完結？ |
|---|------|------|---------------------|
| ① | ネタ指定 | Claude（ネタ帳・トレンドから生成） | ✅ 完結 |
| ② | 台本作成（1分≒300〜350字） | Claude | ✅ 完結 |
| ③ | 自分っぽい口語に直す | Claude＋**文体プロファイル** | ✅ 完結（要: 過去投稿・話し方サンプル） |
| ④ | TTS音声化（声クローン） | **ElevenLabs**（外部API） | △ API呼び出しスクリプトはClaudeが書く |
| ⑤ | 冒頭の実写リップシンク | **HeyGen / Hedra**（外部） | △ 同上（or 手動アップロード） |
| ⑥ | 後半の解説動画生成 | **Remotion**（Claude Codeがコード生成） | ✅ 完結（remotion-best-practices スキルあり） |
| ⑦ | 統合（結合・書き出し） | **ffmpeg**（既にTORU紹介動画で使用実績） | ✅ 完結 |

→ **7工程中5つはClaude Codeで完結。外部依存は④⑤のみ**（どちらもAPI/Web UIあり）。
「後半はClaudeCodeが動画生成（どう作ってるかは知らん）」の正体はほぼ確実に **Remotion**（React で動画を書くフレームワーク。テロップ・図解・アニメーションをコードで量産できる）。

## 2. 外部ツールの選定とコスト

### 音声クローン: ElevenLabs
- 日本語対応（v3で日本語品質が大幅向上・商用OK）
- **Instant Voice Clone**: 音声サンプル1〜3分で即作成（Starter $5/月〜で商用可）
- **Professional Clone**: サンプル30分以上で本人と区別困難レベル（Creator $22/月〜）
- 進め方: まずStarter($5)でInstant Cloneを試す → 品質に納得したらCreatorでPro Clone

### リップシンク実写動画: Hedra → HeyGen の順で検討
- **Hedra**: 自分の写真1枚をアップして音声に合わせて喋らせる方式。無料枠あり・柔軟。**まずこれで試す**
- **HeyGen**: 2分程度の自撮り動画からカスタムアバター作成。品質は最高クラスだが、API利用は$100/月ミニマムで現段階では過剰 → 使うならWeb UI手動（Creator $24〜29/月）
- **Kling**: 静止画/既存動画に新しい音声をリップシンク。コスパ枠

### 月額コスト試算（スタート構成）
| 項目 | 月額 |
|------|------|
| ElevenLabs Starter | $5（Pro Clone化なら$22） |
| Hedra（無料枠→必要なら有料） | $0〜10程度 |
| X Premium+（X記事を書くなら必須） | 数千円 |
| Remotion / ffmpeg | 無料 |
| **合計** | **おおよそ ¥3,000〜8,000/月** |

## 6. X記事（長文記事）ワークフロー

- X記事は **Premium+ 加入者**が書ける長文機能（画像・埋め込み可）。AIで下書き→人間が仕上げ、は完全に可能
- 行動心理を組み込んだ「記事スキル」を作る。チェックリスト案:
  - **フック（1行目）**: 損失回避（「知らないと損」）・好奇心ギャップ（結論を少し見せて引く）・自分事化（読者の状況を名指し）
  - **構成**: PAS（問題→あおり→解決）or 結論先出し→根拠3つ→具体例→CTA
  - **数字の具体性**: 「半日→10分」のような体験ベースの数字（TORUで実証済みの訴求と同じ型）
  - **保存させる設計**: リスト・手順・テンプレ配布（保存＝アルゴリズム評価が高い）
  - **改行リズム**: スマホ1画面に1メッセージ
- 自分の実体験（配管屋×AI×個人開発）が最強の差別化。一般論記事はAIで誰でも書けるので、**体験×データだけは本人しか出せない**

## 7. サムネ・バナー生成 → **作れる。方式は2つ**

1. **HTML/CSS→Playwrightスクショ方式（推奨・メイン）**
   - Claude CodeがHTMLでバナーをデザイン → ヘッドレスブラウザで1200x675/1080x1920のPNGに撮影
   - **日本語テキストが絶対に崩れない**（画像生成AIは日本語文字がまだ苦手）
   - ブランド一貫性（色・フォント・ロゴ固定）が保てる。TORUのBlueprintデザイン資産も流用可
   - コスト0円・完全自動化可
2. **画像生成API方式（補助）**
   - 背景イラスト・イメージカットだけ画像生成APIで作り、文字はHTML側で載せるハイブリッドが現実解

## 8. スキル化計画（この期間に作る資産）

| スキル名（案） | 中身 | 価値 |
|---------------|------|------|
| `script-factory` | ネタ帳→台本→口語化（文体プロファイル内蔵） | 動画・投稿の上流を完全自動化 |
| `thumbnail-banner` | HTML/CSS→PNGバナー（テンプレ複数） | 動画サムネ・記事ヘッダー・LP OGP全対応 |
| `x-article-writer` | 行動心理チェックリスト込みの記事生成 | インプ獲得の再現性 |
| `shorts-video-builder` | ElevenLabs API＋Remotion＋ffmpeg統合 | 「1コマンドで1本」の本体 |
| `buzz-post-analyzer` | バズ投稿の構造分解→自分ネタへの転用 | ネタ切れ防止・勝ちパターン学習 |

**スキルの将来性**: スキルはモデルが変わっても動く「手順書資産」。さらに「AIで事業をやりたい人向けのClaude Code活用スキル」は、それ自体が発信ネタ・商材になる（海外でn8nテンプレ販売が成立しているのと同じ構図）。TORUで実際に使った営業・開発の手順をスキル化すれば「本物の実績つき」で出せる。
