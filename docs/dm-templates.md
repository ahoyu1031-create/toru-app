# DM君 — X コールドDM テンプレ集

TORU を試してもらう / 現場の声を聞くための、コピペ用DMテンプレ。
最終更新: 2026-06-03。URL: `toru-app.vercel.app`

> 目的は「売る」より **「現場・経営のリアルを教えてもらう＋試してもらう」**。
> 最初の数十件は **学習**（痛みが設備以外にも共通するか／経営目線で刺さるか）が本命。

---

## 🔑 使い方（鉄則・送る前に必ず読む）

1. **送る前に相手の投稿へ いいね/リプを1〜2回**。非フォロワーへのDMは「リクエスト」に埋もれて気づかれない。先に認知させる。
2. **冒頭1行は必ず個別化**（`〇〇` を相手の会社名・投稿内容に差し替え）。同一文面の連投は**スパム判定＆相手にバレる**。
3. **1日2〜3通まで**。若いアカウントで一気に送ると凍結リスク。
4. **トーンは「教えてください／意見ほしい」**。返信ハードルを下げる。「使ってください」連打はしない。
5. **URLは原則2通目**（返信が来てから）。※社長相手だけは時間が無いので簡潔ワンショット（URL入り）も可。
6. **記録をつける**（下のトラッキング表）。5〜10件でパターンが見えたら打ち手を変える。
7. **数字は盛らない／嘘つかない**。個人開発の体温（正直さ）が一番の武器。

---

## 🧩 営業DM 構文（5パート・量産用）

テンプレ丸コピーじゃなく、この5パートを上から組んで **1行目だけ必ず個別化**する。

1. **掴み（個別化）** — 相手の投稿/プロフィールに1行触れる。「あなた宛て」の証明。※ここだけは絶対コピペしない
2. **刺し（痛み）** — 痛みを質問で1行。「〜時間取られてませんか？」
3. **正体＋価値** — 何者・何を・価値（半日→10分）。★ここに殺し文句（下）
4. **小さなお願い** — 「買って」じゃなく「5分見て/試して/感想ください」
5. **逃げ道** — 「お忙しければスルーで🙏」。圧を消す＝返信率UP

### ★殺し文句：「現場の配管屋が作ってる」
AI開発者が作ったツール ＜＜＜ **同じ現場の配管屋が作ったツール**。建設の人の信用度が桁違い。
最大の差別化はコードじゃなく「当事者であること」。**毎回これを匂わせる。**（事実なので堂々と）

### 組み立て例（→ 1行目だけ差し替えて量産）
```
〇〇さんの投稿、めちゃ共感して連絡しました🙇

図面の拾い出し〜見積、今かなり時間取られてませんか？

その作業をAIで半日→10分にするツールを、
自分も現場の配管屋やりながら個人で作ってます。

もしピンと来たら、5分触って感想だけもらえたら嬉しいです。
（お忙しければスルーで🙏）
→ toru-app.vercel.app
```

### ゲリラ運用のガードレール（アカBANを避ける）
- **1行目だけは必ず個別化**。同一文面の連投＝スパム判定＆相手にバレる
- 完全cold DMは「メッセージリクエスト」に埋もれる → 送る前に相手の投稿へ **いいね/リプ1個**で認知
- **DM開いてる人だけにDM**。閉じてる人はリプで攻める（公開・権限不要・相手のフォロワーにも見える＝高効率）
- **1日5〜10通から**。若いアカで一気に大量＝凍結リスク。様子見て増やす
- 記録 → 5〜10件で「どの一言が返るか」を見て調整
- ※現状は **DMよりリプの yield が高い**（DM権限の壁が無い）。DMはDM開放アカに絞り、リプと並行が最効率

## A. 社長・経営者向け ★今日のメイン

経営者は「人件費・残業・受注スピード・属人化」で動く。現場作業の時間より**“いくら浮くか／受注が早まるか”**で語る。

### A-① オープナー（URLなし・1通目）

```
突然のDM失礼します🙇 〇〇（会社/投稿）拝見しました。
建設の経営目線で一つ伺いたくて——

御社では、図面からの材料拾い出し・見積作成に、担当の方どれくらい時間かけてますか？

その作業をAIで自動化するツールを個人で作っていて、経営側から見て本当に刺さるか確かめたいんです。一言だけでもご意見もらえたら嬉しいです。お忙しければ流してください🙏
```

### A-② 返信が来たら（2通目・価値＋無料）

```
ありがとうございます、めちゃ参考になります！
それを自動化するのがこれです → toru-app.vercel.app

図面PDFを入れると、材料の拾い出し〜見積書PDFまで自動。
半日〜1日かかっていた作業が、10分に。

無料・クレカ不要で試せます（チーム単位でも使えます）。
担当の方に5分だけ触ってもらって、率直な感想いただけたら開発の超ヒントになります🙏
```

### A-③ 簡潔ワンショット（多忙な経営者に一発で・URL入り）

```
突然失礼します🙇 〇〇拝見しました。
図面からの材料拾い出し〜見積作成を自動化するAIを個人で作っています → toru-app.vercel.app

半日〜1日の作業が10分に。無料・クレカ不要で試せます。
現場の経営目線でご意見いただけたら嬉しいです（お忙しければ流してください）
```

## ⚠️ 禁止事項

- 同一文面の一斉送信（個別化必須）
- 1通目からURL連打 + 「買ってください」（売り込み臭＝即ブロック）
- 1日に大量送信（凍結リスク／1日2〜3通）
- 数字の誇張・嘘（致命傷）
- 返信が無くても催促DMを送らない
