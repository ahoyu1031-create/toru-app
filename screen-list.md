ワイヤーフレーム作成前の画面棚卸し。Phase別に整理。

作成: 2026-04-19

## 概要

各画面に「目的・主要要素・遷移」を1行で添える。
v0.1〜v0.3の段階リリースに合わせて画面を分けて管理する。

---

## 設計・方針

- **3タップで完結** が基本（経営者が迷わない）
- シニア層対応: 大きめタップ領域・大きめフォント
- **露骨さと上品さのバランス**: 大きめUIでも絵文字の過剰使用や装飾過多は避ける。洗練されたミニマル路線（Notion系の余白感 × freee系の業務UI）で「わかりやすい ＝ 不格好」にならないよう注意
- ダーク/ライト両対応
- Web版とモバイル版でナビゲーション構造を揃える
  - モバイル: タブバー（下部）
  - Web: サイドバー（左）
- **Web版先行リリース** → モバイル版並行開発

## UIトーンガイド（シニア配慮×上品さ）

| やる | やらない |
|------|---------|
| 大きめフォント（本文16px+、ボタンラベル18px+） | 絵文字ゴテゴテ（📝🚀✨ の乱発） |
| 十分な余白・コントラスト | 派手なグラデーション・ネオンカラー |
| 主要ボタンは太めで明確（アイコン+短文） | 装飾的な影・ベベル |
| 色は機能で使い分け（警告は赤、成功は緑） | 意味のない色使い |
| コピーは敬語かつ簡潔 | 砕けすぎた口語・若者言葉 |

#### 拾い出し → 見積書CTA の UI 方針

**シニア配慮×上品さのバランス設計:**

```
┌────────────────────────────────────────┐
│ 拾い出し結果 - 現場A                     │
│                                        │
│  材料          単位    数量             │
│  ───────────────────────────          │
│  塩ビ管 VU50   m       12              │
│  エルボ 50A    個      8               │
│  ...                                   │
│                                        │
├────────────────────────────────────────┤
│                                        │
│  ┌──────────────────────────────┐     │
│  │  この結果から見積書を作成する   │     │  ← 大きめ、シンプル
│  └──────────────────────────────┘     │     色: プライマリ
│                                        │
│  または [CSVで書き出す] [編集する]     │     ← サブ導線
└────────────────────────────────────────┘
```

- **メインCTA**: プライマリカラーの大きめボタン、アイコンは使わずラベルだけで勝負
- 絵文字は使わない（「📝」などでチープにしない）
- サブアクション（CSV書き出し・編集）は控えめテキストリンク
- ホーム画面では「未変換の拾い出しが X件」の軽めバッジで想起、**プッシュ通知はデフォルトOFF**

### v0.3: グループ機能

**アクセス制限:** グループ**作成**は法人プラン（team_5/team_10/team_unlimited）のみ。個人プランは招待経由の**参加**のみ可。

| 画面 | 目的 | プラン制限 |
|------|------|-----------|
| グループ一覧 | 参加中のプロジェクト/現場グループ | 全プラン |
| **グループ作成** | グループ名・説明・**信頼レベル**（trusted/mixed）選択 | 法人のみ |
| グループ詳細 | 共有アイテム一覧・メンバー一覧 | 全プラン |
| **メンバー招待画面** | 招待リンク発行・QRコード・メール招待、**コピーボタン付き** | 法人のみ |
| **グループID手入力参加** | 6桁のグループIDを入力して参加申請 | 全プラン |
| 参加申請承認 | 作成者が申請一覧を確認し、1タップで承認/却下 | 法人のみ |
| 共有実行 | コンテンツ選択 → **3段階確認モーダル**（→ `group-share-confirmation-spec.md`） | 全プラン |
| 共有取り消し確認 | 「閲覧履歴は残りますがいいですか？」 | 全プラン |
| 共有ログ表示 | グループ内の共有履歴・誰がいつ何を共有したか | 全プラン |

```
┌───────────────────────────────────────┐
│ メンバーを招待                         │
│                                       │
│ グループID: TK7P2M                     │
│                                       │
│ ┌─────────────────────────────┐      │
│ │ 招待リンク                    │      │
│ │ https://toru.app/j/TK7P2M    │      │
│ │                 [📋 コピー]   │      │ ← ワンタップでクリップボード
│ └─────────────────────────────┘      │
│                                       │
│ [LINEで送る] [メールで送る]           │ ← 主要アプリへの直接共有
│                                       │
│ ┌─────────────────────────────┐      │
│ │     [QRコード表示]            │      │ ← 対面招待用
│ └─────────────────────────────┘      │
│                                       │
│ または メールアドレスで招待:          │
│ [____________________] [招待する]    │
└───────────────────────────────────────┘
```

- **コピーボタン**を招待リンク横に大きめ配置（LINE等への転送用）
- LINEショートカットボタンで共有UIを直接起動
- QRコードは対面時用（現場ミーティング等）
- グループID（TK7P2M）は電話でも伝えやすい6桁英数字
