# 単価マスタ機能 — 仕様書

## 概要

会社ごとに異なる材料単価を管理する専用ページ。
図面解析で拾い出した材料にこのマスタを適用して見積書を自動生成する。

---

## データ構造

### UnitPriceMaster（単価マスタ）

```
company_id    : string   -- 会社ID（会社ごとに独立）
material_name : string   -- 材料名（例: 塩ビ管 VU50）
unit          : string   -- 単位（例: m, 本, 個, セット）
unit_price    : number   -- 単価（円）
category      : string   -- カテゴリ（任意: 配管材, 継手, 弁類 等）
memo          : string   -- メモ（任意）
created_at    : datetime
updated_at    : datetime
```

---

## 画面仕様

### 単価マスタ一覧ページ
- 登録済み材料の一覧表示
- 材料名・単位・単価を表形式で表示
- カテゴリでフィルタリング
- 検索（材料名）
- 編集・削除ボタン

### 単価登録・編集ページ
入力フォーム:
1. **材料名** (必須) — テキスト入力
2. **単位** (必須) — テキスト or セレクト（m / 本 / 個 / セット / 式）
3. **単価** (必須) — 数値入力（円）
4. **カテゴリ** (任意) — テキスト or セレクト
5. **メモ** (任意) — テキスト

---

## 見積書生成フロー

```
図面解析結果
    ↓
拾い出し材料リスト
（material_name + quantity）
    ↓
単価マスタで material_name をマッチング
    ↓
小計 = unit_price × quantity
    ↓
合計金額算出
    ↓
見積書PDF出力
```

### マッチング未対応材料の処理
- 単価マスタに存在しない材料は「未登録」として表示
- 手動で単価を入力するか、マスタに追加してから再計算

---

## 権限設計（案）

| 権限 | できること |
|------|-----------|
| 管理者（経営者） | マスタの全操作 + 見積書作成 |
| 一般（作業員） | 図面解析のみ（マスタ参照はOK、編集不可） |
